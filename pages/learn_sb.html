SRO: Server Browser System

Discover servers. Sync clients. Connect instantly.

🧭 SRO:SB – Server Browser

This modern and lightweight system is designed to help players find, synchronize, and connect to Silkroad Online private servers effortlessly. Whether you're a server admin or a player, SRO:SB provides a seamless experience.

📦 System Components

🔹 SRO:SB – Server Browser

For Players: A desktop client that provides real-time server listings with detailed server metadata such as ping, player count, description, language, website, and more. Enjoy delta synchronization, which allows you to download only modified files, reducing data usage and sync time.

🔹 SRO:SH – Server Hub

For Server Admins: A lightweight background service that communicates with your private server and sends updates to a centralized hub. It supports SQL database integration for remote account creation.

🔹 SRO:CD – Client Database

For Synchronization: An open-source GitHub repository that stores Silkroad client modifications. Players only download the differences, saving bandwidth and time.

🌍 Why This Project?

The Silkroad private server community is vibrant but fragmented. This project addresses key issues:

  • Centralized, reliable list of active servers
  • Clear server status and metadata
  • Eliminating the need for constant full client downloads
  • Automation in account creation and patching

⚙️ Technical Overview

Built using Python's asyncio for non-blocking I/O, with TCP communication using JSON payloads. Optimized for low bandwidth and processing power, this system is fully operational on Android-based Linux systems (e.g. Termux).

🧪 Status

The core system is fully functional. A GUI-based version is currently under development.

🧭 SRO:Sunucu Tarayıcı Sistemi

Silkroad Online özel sunucularını keşfetmek, senkronize etmek ve bağlanmak için geliştirilen modern ve hafif bir altyapı. Bu proje, oyuncular ve sunucu sahipleri için süreçleri daha verimli hale getirmeyi hedefliyor.

📦 Sistem Bileşenleri

🔹 SRO:SB – Server Browser

Oyuncular için: Gerçek zamanlı sunucu listeleri ve ping, oyuncu sayısı, açıklama, dil, web sitesi gibi sunucu bilgilerini sağlar. Ayrıca sadece değiştirilen dosyaları indirmenizi sağlayarak veri kullanımını ve senkronizasyon süresini azaltan delta senkronizasyon özelliği sunar.

🔹 SRO:SH – Server Hub

Sunucu sahipleri için: Sunucu yazılımı ile birlikte çalışan hafif bir arka plan hizmetidir. Merkezi bir HUB'a düzenli güncellemeler gönderir. İsteğe bağlı olarak SQL veritabanı ile kullanıcı oluşturma desteği sunar.

🔹 SRO:CD – Client Database

Referans ve senkronizasyon için: Silkroad istemci modifikasyonlarını barındıran açık kaynaklı bir GitHub deposudur. Oyuncular sadece değişen dosyaları indirir, böylece bant genişliği ve zaman tasarrufu sağlanır.

🌍 Neden Bu Sistem?

Silkroad özel sunucu topluluğu canlı ancak dağınıktır. Bu sistem şu sorunları çözmeyi hedefler:

  • Aktif sunucuların merkezi ve güvenilir listesi yok
  • Sunucu durumları ve bilgileri belirsiz
  • Sürekli baştan istemci indirme zorunluluğu
  • Hesap oluşturma ve yama işlemlerinde otomasyon eksikliği

⚙️ Teknik Özellikler

Python (asyncio) kullanılarak eş zamanlı ve bloklamayan bir yapı ile geliştirilmiştir. TCP + JSON veri alışverişi yapılır. Bant genişliği ve işlemci yükü düşük ortamlarda çalışacak şekilde optimize edilmiştir.

🧪 Geliştirme Durumu
